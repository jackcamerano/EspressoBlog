# Getting Started

EspressoBlog is a next template based on [Blogify](https://github.com/frontendweb3/blogify)

## Start with blogify

```bash

## Step 1

git clone git@github.com:giacomocamerano/jackcamerano.com.git

## Step 2
cd jackcamerano.com

## Step 3
pnpm install
# or
yarn install

## Step 4
pnpm dev
# or
yarn dev
```
